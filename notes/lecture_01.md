## 2.Bash commands
### Short keys
Extended cursor:
SHIFT + CTRL = ALT + arrow up/down
Copy text: CTRL + C
Cut text: CTRL + X
Paste text: CTRL + V
Copy all row: SHIFT + ALT + arrow up/down
Comment line: CTRL + /
Undo: CTRL + Z
Move row/s: ALT + arrow up/down
Choose previous command within terminal: arrow up/down
Save file: Cntr + S
